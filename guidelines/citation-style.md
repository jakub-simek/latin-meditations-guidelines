# Citation Style

## Direct Quotations

- Direct quotations are italicized.
- Do not use quotation marks in the Latin meditation text.
- Place the source reference in parentheses directly after the quotation.
- When the source reference immediately follows the quotation, omit the final
  period inside the italic quotation and place the sentence period after the
  parenthetical reference.
- Omissions within quotations use the typographic ellipsis character with spaces
  on both sides: `verbum ... verbum` in plain examples, and the true ellipsis
  character in final meditation text.
- Direct quotations must be real, traceable, and substantially faithful to a
  reliable source.
- Prefer short, luminous quotations over long documentary blocks.

## Primary Text References

- In the Textus fundamentalis, give the full reference required by the project
  for the primary text.
- Later quotations from the same primary text may normally use a concise
  locator, unless the source context would otherwise be unclear.
- The project overlay defines the exact locator form: biblical locus, chapter
  and section, sermon number, paragraph, page, or another stable reference.
- When quoting another text that is not the primary meditation text, give the
  fuller reference required by the project.

## Paraphrase and Indirect Reference

- Paraphrases are not italicized.
- Use `cf.` in the source reference.
- Use paraphrase when summarizing an argument, drawing a thematic connection, or
  when exact wording has not been verified sufficiently for direct quotation.

## Source Titles and References

- Source references should be typographically distinct from the quoted text.
- Do not italicize the whole reference merely because it follows an italicized
  quotation.
- Titles within references may be italicized according to the project's normal
  style.
