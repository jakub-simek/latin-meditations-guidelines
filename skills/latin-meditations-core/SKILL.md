# Latin Meditations Core

## Source of Truth

Read the relevant shared guidelines before working on a meditation:

- `../../guidelines/general-meditation-principles.md`
- `../../guidelines/latin-style.md`
- `../../guidelines/citation-style.md`
- `../../guidelines/source-verification.md`
- `../../guidelines/review-checklist.md`

Then read the local project overlay. Project-specific rules override these
general rules when they are stricter or more concrete.

## Workflow

1. Identify the primary text and the project-specific source witness.
2. Read the local project overlay for metadata, required source families, and
   citation conventions.
3. Check related meditations or project files when the overlay requires it.
4. Write or revise the meditation according to the shared guidelines and local
   overlay.
5. Verify direct quotations, `cf.` references, source locators, and original
   language rules.
6. Make a final Ecclesiastical Latin correction pass.

## Review

Use `../../guidelines/review-checklist.md` together with the local project
checklist.
